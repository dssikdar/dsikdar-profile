# Diptanshu Sikdar Academic Profile

Static academic profile site for GitHub Pages.

## Publish on GitHub Pages

1. Create a repository named `dssikdar.github.io` or enable Pages on this repository.
2. Commit `index.html`, `styles.css`, `assets/`, and `files/`.
3. In GitHub, set Pages to deploy from the main branch root.

The site does not require a build step.
